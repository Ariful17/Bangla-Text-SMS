# Bangla-Text-SMS
This is a project using Arduino and GSM module for receiving Bangla Text with the help of an oled display. <br>
- IDE- Arduino<br>
- Board- Arduino Mega 2560<br>
- GSM Module- Sim800L<br>
- Display- 0.96" oled display<br>
